About

I am a Technical Writer and I created this blog to see how a static site generator might used for documentation projects.

I got the idea to try this out after reading this I’d Rather be Writing article.

I have the blog now, but still looking into how this might be used for documentation. There are a lot of pieces to these things! When I have time I’d like to learn more about pdf generators, node.js, responsive design, and several hundred other things that might be helpful.

It’s a work in progress!
