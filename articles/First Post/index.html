This project uses the Wintersmith static web site generator. My goals for doing this project were to:

  1. Learn how to host a free web site on Github.
  2. Learn how Markdown might be used to author documents.
  3. Learn how a static web site generator that uses Markdown works.
  4. Update my CSS skills.
  5. Experiment with responsive grids.

Many thanks to the developer of Wintersmith, as well as:

  • Simple Grid
  • YearlongHackathon (for explaining how to add the partials section for Jade that I was able to use to auto generate the menu sidebar).

and so many other articles that were read in the google browsathon that it took to get this initial web site up!
