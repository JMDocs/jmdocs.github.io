Automate with Grunt

Oct 20, 2015

My Wintersmith blog is now formatted more or less as I want it. Will likely tweak it here and there. I haven’t tested extensively, but have tried it on Chrome, Firefox, Safari, Windows Phone. Seems okay. (The minute I write that I’ll notice something to change.)

So now on to the next hurdle. Simplifying the updates. At the moment, after making a change, I rebuild the site on my computer, then use the following git commands to upload the changes to GitHub:

git add . git commit -a -m "this is the message" git push Enter username and password.

That’s not too arduous! But it would be nice to streamline. It sounds like if I was using Jekyll and Github pages this would be done automatically. But as I’m not using Jekyll, I’ll need to set up my own system. With the added bonus of learning more about Grunt.
